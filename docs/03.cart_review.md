# 장바구니 페이지 리뷰

## 고객사 요구사항
장바구니 페이지는 사용자가 쇼핑 과정에서 선택한 상품을 한눈에 확인하고, 주문을 진행할 수 있도록 하는 화면입니다. 고객사에서 전달한 요구사항과 디자인 시안을 기준으로 다음과 같은 기능을 구현하였습니다.

-사용자가 장바구니에 담은 상품 목록을 확인할 수 있어야 한다.
-각 상품에는 상품 이미지, 상품 이름, 가격 정보가 표시되어야 한다.
-사용자는 장바구니에 담긴 상품을 삭제할 수 있어야 한다.
-장바구니에 상품이 없을 경우 빈 장바구니 상태 화면이 표시되어야 한다.
-사용자는 장바구니 페이지에서 결제하기 버튼을 통해 다음 단계로 이동할 수 있어야 한다.
-결제가 불가능한 상황(예: 장바구니가 비어 있는 경우)에는 결제 버튼이 비활성화 상태로 표시되어야 한다.
-전체 화면은 사용자가 상품을 쉽게 확인하고 결제할 수 있도록 직관적인 구조로 구성되어야 한다.

피그마 디자인 시안에 맞추어 장바구니 페이지의 레이아웃과 주요 UI 요소를 구현하였습니다.

## 요구사항에 따른 기능 동작 확인

아래는 실제 구현된 기능과 그 동작 상태를 확인한 내용입니다.

### 1. 장바구니 상품 목록 표시

사용자가 장바구니에 상품을 추가하면 해당 상품이 장바구니 페이지에 목록 형태로 표시됩니다.
각 상품에는 다음 정보가 함께 표시됩니다.

- 상품 이미지
- 상품 이름
- 상품 가격

사용자는 장바구니 페이지에서 현재 선택된 상품들을 한눈에 확인할 수 있습니다.

#### 동작 상태

정상적으로 동작합니다.
장바구니에 담긴 상품이 목록 형태로 표시되며 상품 정보가 올바르게 나타납니다.

### 2. 장바구니 상품 삭제 기능

사용자는 장바구니에 담긴 상품을 삭제할 수 있습니다.
상품 삭제 버튼을 클릭하면 해당 상품이 장바구니 목록에서 제거됩니다.

#### 동작 상태

정상적으로 동작합니다.
삭제 버튼을 클릭하면 해당 상품이 즉시 장바구니 목록에서 사라집니다.

### 3. 장바구니가 비어 있는 경우 안내 화면

장바구니에 상품이 없는 경우에는 상품 목록 대신 빈 장바구니 안내 화면이 표시됩니다.
이를 통해 사용자는 현재 장바구니에 담긴 상품이 없다는 것을 쉽게 알 수 있습니다.

#### 동작 상태

정상적으로 동작합니다.
장바구니에 상품이 없을 경우 빈 상태 안내 화면이 표시됩니다.

### 4. 결제하기 버튼

장바구니 페이지 하단에는 결제하기 버튼이 배치되어 있습니다.
사용자는 이 버튼을 통해 주문 또는 결제 단계로 이동할 수 있습니다.

#### 동작 상태

정상적으로 동작합니다.
버튼을 클릭하면 결제 단계로 이동하는 동작이 실행됩니다.

### 5. 결제 버튼 비활성화 상태

장바구니에 상품이 없는 경우에는 사용자가 결제를 진행할 수 없기 때문에 결제하기 버튼이 비활성화 상태로 표시됩니다.

이를 통해 사용자가 잘못된 상태에서 결제를 진행하는 것을 방지합니다.

#### 동작 상태

정상적으로 동작합니다.
장바구니가 비어 있을 경우 결제 버튼이 비활성화 상태로 표시됩니다.

### 6. 사용자 편의성을 고려한 화면 구성

장바구니 페이지는 사용자가 상품 정보를 빠르게 확인하고 결제를 진행할 수 있도록 단순하고 직관적인 구조로 구성되어 있습니다.

상품 목록, 삭제 기능, 결제 버튼이 한 화면에서 자연스럽게 이어지도록 배치하여 사용자의 이용 편의성을 고려하였습니다.

#### 동작 상태

디자인 시안과 동일한 구조로 화면이 구성되어 있으며 사용성에 문제가 없습니다.

현재 장바구니 기능은 기본적인 사용자 흐름을 기준으로 구현되었습니다.
향후 서비스 확장을 위해 다음과 같은 기능을 추가할 수 있습니다.

- 총 결제 금액 표시
- 배송비 계산
- 실제 결제 시스템 연동

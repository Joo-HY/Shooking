# 장바구니 페이지 요구사항 분석 및 기능 목록

- React
- JavaScript
- Vite
- Tailwind CSS
- React Router
- 기존 결제 모듈 연동 로직

## 📝 고객 요구사항 정리
### 1. 장바구니 상품 목록 표시

- 현재 장바구니에 담긴 상품 목록을 표시한다.

- 각 상품에는 다음 정보가 포함된다.
  - 상품 이미지
  - 브랜드명
  - 상품 가격
  - 수량 조절 버튼

- 상단에는 현재 담긴 상품 개수를 표시한다.
  - 예시 : 현재 3개의 상품이 담겨있습니다.

### 2. 수량 조절 기능

- 각 상품마다 수량 증가(+) / 감소(-) 버튼을 제공한다.
- 수량 변경 시 해당 상품의 금액과 전체 금액이 즉시 다시 계산되어야 한다.

### 3. 상품 금액 계산
- 장바구니에 담긴 모든 상품의 가격을 합산하여 상품 금액을 계산한다.
- 계산 방식
  - 상품 금액 = Σ (상품 가격 × 수량)

### 4. 배송비 정책
- 고객사 이메일 기준 배송비 정책
  - 총 상품 금액 100,000원 미만 : 배송비 3,000원
  - 총 상품 금액 100,000원 이상 : 배송비 무료

### 5. 총 결제 금액 표시
- 장바구니 하단에 다음 금액 정보를 표시한다.
  - 상품 금액
  - 배송비
  - 총 금액 : 총 금액 = 상품 금액 + 배송비

### 6. 뒤로가기 기능
- 페이지 상단 좌측에 뒤로가기 버튼을 배치한다.
- 버튼 클릭 시 : 이전 페이지로 이동
- React Router 기준 : navigate(-1)

### 7. 결제하기 기능
- 페이지 하단에 결제하기 버튼을 배치한다.
- 버튼 클릭 시 :
  - 이전에 구현한 결제 모듈을 호출
  - 장바구니 상품 정보 및 총 금액을 전달

## 📋 기능 목록
### 1. 장바구니 페이지 레이아웃 구성
- 기능 설명 : 장바구니 페이지 전체 UI 구조를 구현한다.
- 세부 작업
  - 페이지 헤더 영역 생성
  - 뒤로가기 버튼 구현
  - 페이지 제목 표시
  - 장바구니 상품 개수 표시
  - 상품 목록 영역 생성
  - 금액 요약 영역 생성
  - 결제 버튼 영역 생성
- 사용 컴포넌트
  - CartPage
  - CartHeader
  - CartItemList
  - CartSummary
  - CheckoutButton
- 예외 처리
  - 장바구니가 비어 있을 경우 별도 UI 표시
  - 상품 개수가 0일 때 자연스러운 문구 출력

### 2. 장바구니 상품 목록 표시
- 기능 설명 : 장바구니에 담긴 상품 데이터를 리스트 형태로 렌더링한다.
- 세부 작업
  - 장바구니 데이터 구조 설계
  - 상품 리스트 반복 렌더링(map)
  - 상품 이미지, 브랜드명, 가격 표시
  - 각 상품별 수량 조절 UI 생성
- 사용 컴포넌트
  - CartItemList
  - CartItem
- 예외 처리
  - 상품 이미지가 없는 경우 기본 이미지 표시
  - 상품 데이터가 null일 경우 렌더링 오류 방지

### 3. 상품 수량 조절 기능
- 기능 설명 : 상품 수량을 증가 또는 감소할 수 있도록 한다.
- 세부 작업
  - 버튼 클릭 시 수량 증가
  - 버튼 클릭 시 수량 감소
  - 수량 변경 시 금액 자동 재계산
- 사용 컴포넌트
  - QuantityControl
  - CartItem
- 예외 처리
  - 수량이 1 이하로 내려가지 않도록 제한
  - 비정상적인 수량 값 방지
 
### 4. 상품 금액 합계 계산
- 기능 설명 : 장바구니 상품들의 총 금액을 계산한다.
- 세부 작업
  - 상품 금액 = 가격 × 수량
  - 전체 상품 금액 = 모든 상품 금액 합계
  - 수량 변경 시 자동 계산
  - 화면에 실시간 반영
- 사용 컴포넌트
  - CartSummary
- 예외 처리
  - 가격이 숫자가 아닌 경우 변환 처리
  - NaN 발생 방지

### 5️. 배송비 계산 기능
- 기능 설명 : 상품 금액에 따라 배송비를 계산한다.
- 세부 작업
```
if (상품금액 < 100000)
배송비 = 3000
else
배송비 = 0
```
- 사용 컴포넌트
  - CartSummary
- 예외 처리
  - 장바구니가 비어 있을 경우 배송비 0원 처리

### 6. 총 결제 금액 계산
- 기능 설명 : 상품 금액과 배송비를 합산하여 총 금액을 표시한다.
- 세부 작업
  - 총 금액 = 상품 금액 + 배송비
- 사용 컴포넌트
  - CartSummary
- 예외 처리
  - 계산 오류 발생 시 0원 처리
### 7. 뒤로가기 기능
- 기능 설명 : 사용자가 이전 페이지로 이동할 수 있도록 한다.
- 세부 작업
  - 뒤로가기 버튼 클릭 이벤트 구현
  - React Router 활용
- 사용 컴포넌트
  - BackButton
  - CartHeader
- 예외 처리
  - 이전 페이지가 없는 경우 홈 페이지로 이동

### 8. 결제하기 버튼 기능
- 기능 설명 : 결제 버튼 클릭 시 결제 모듈을 호출한다.
- 세부 작업
  - 결제 버튼 UI 구현
  - 장바구니 상품 정보 전달
  - 총 결제 금액 전달
  - 결제 모듈 호출
- 사용 컴포넌트
  - CheckoutButton
  - CartPage
- 예외 처리
  - 장바구니가 비어 있을 경우 결제 불가
  - 결제 실패 시 오류 메시지 표시

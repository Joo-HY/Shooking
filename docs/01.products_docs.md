# 상품 목록 페이지 프로젝트

## 프로젝트 개요

본 프로젝트는 사용자가 다양한 상품을 한눈에 확인하고, 상품의 기본 정보를 탐색할 수 있도록 구성한 상품 목록 페이지 구현을 목표로 한다.

주요 기능:
- 상품 리스트 동적 렌더링
- 상품 이미지, 이름, 가격 정보 표시
- 상품 상세 페이지 이동 또는 장바구니 기능 확장 구조 설계
- 컴포넌트 기반 구조 설계

참여 인원:
- 완두앙금빵

## 개발 환경

- 프레임워크 : React
- 빌드 툴 : Vite
- 언어 : JavaScript
- 스타일링 : Tailwind CSS
- 테스팅 : Vitest
- 패키지 매니저 : npm
- Version Control : Git / GitHub
- Deployment : GitHub Pages

## 설치 및 실행 방법

프로젝트를 로컬에서 실행하기 위한 설치 방법과 명령어

```
git clone https://github.com/Joo-HY/Shooking.git
cd shooking
npm install
npm run dev
http://localhost:5173/
```

## 컴포넌트 구조

주요 컴포넌트들의 구조와 역할

- `ProductList`: 상품 배열을 받아 반복 렌더링. ProductCard를 map 함수로 출력.
- `ProductCard`: 개별 상품의 이미지, 이름, 가격을 표시. 버튼 이벤트 처리 (예: 상세보기 또는 장바구니 추가)

## 테스트 방법

테스트 케이스 및 실행 방법

```
npm run test
```

## 유의사항 및 알려진 이슈

- 모바일 환경에서 일부 레이아웃이 깨질 수 있음 (반응형 보완 필요)
- 상품 데이터가 현재는 Mock 데이터로 구성되어 있음 (추후 API 연동 예정)
- 상태 관리가 단순 구조로 되어 있어 확장 시 Context 또는 Redux 도입 고려 필요

## 추가 항목

기술 선택 이유
- Vite : 빠른 개발 환경과 빌드 속도를 위해 사용
- Tailwind CSS : Utility-first 접근을 통해 일관된 스타일 관리 및 빠른 UI 구현을 위해 사용
- Vitest : Vite 환경과 통합된 경량 테스트 환경 구성을 위해 사용
